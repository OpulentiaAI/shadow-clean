# Instructions

Convert any web URL to clean markdown content. This is a lightweight tool optimized for quickly extracting readable content from webpages.

Use this tool when:
- You only need the text content of a page in markdown format
- You want a quick, lightweight read of a webpage
- You're extracting documentation or article content

## When to use this vs web_read_url

- Use `web_get_markdown` when you only need the text content
- Use `web_read_url` when you also need metadata, title, or structured information about the page

## Tips

- This is the fastest way to read webpage content
- Results are cached for repeated requests
- Ideal for documentation pages, articles, and blog posts
- For GitHub READMEs, this extracts clean markdown content
